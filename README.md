### The Three Broomsticks ‚Äì Console Billing System  
A beginner C++ restaurant billing project inspired by the wizarding world

#### About the Project

This was my **very first proper programming lab assignment** (CSC284 ‚Äì Programming in C++, Summer 2023) when I was a 1st-year student in the Department of Computer Science & Engineering at IUBAT.

As a huge Harry Potter fan, I chose **The Three Broomsticks** ‚Äî the famous pub in Hogsmeade ‚Äî as the theme for my imaginary restaurant.  
The task was to build a console program that takes customer orders from a menu, calculates the total with 5% VAT, prints a clean payment slip, and returns change in Bangladeshi Taka (notes + cents).

#### Features

- Hogwarts-style welcoming interface with centered text
- Display of 6 menu items with prices in BDT
- Personalized greeting using customer name
- Order multiple items by serial number + quantity (servings)
- Automatic calculation: subtotal + 5% VAT
- Delivery address collection
- Change returned in Bangladeshi currency notes (1000 ‚Üí 1 TK) + cents
- Basic input validation
